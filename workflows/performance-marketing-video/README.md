# Performance Marketing Video Workflow

AI 도구를 활용한 퍼포먼스 마케팅 영상 제작 파이프라인.

## 개요

이 워크플로우는 AI 영상 생성 도구들을 조합하여, 사람 없이도 고품질의 퍼포먼스 마케팅 영상(UGC, 제품 광고, 소셜 콘텐츠)을 제작하는 end-to-end 파이프라인을 정의합니다.

## 핵심 파이프라인

```
기획/스크립트 → 캐릭터/제품 이미지 생성 → 영상 생성 → 음성/오디오 → 업스케일 → 편집/내보내기
```
