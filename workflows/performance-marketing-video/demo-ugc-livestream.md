## Phase 2: Nano Banana Pro — 구도 변형

> Soul 2.0에서 생성한 Shot A를 레퍼런스 이미지로 사용

### 변형 1 — 폰을 카메라에 보여주는 동작

```
이 여성이 모든 것 그대로 유지한 채,
카메라 방향으로 아이폰을 들어 보여주는 모습,
아이폰 화면에는 인스타그램 라이브 화면이 켜져 있고 하트 이모지가 떠오르는 중,
표정은 살짝 설레는 미소, 자연스러운 손 각도
```

### 변형 2 — 음료를 마시는 일상 동작

```
이 여성이 모든 것 그대로 유지한 채,
책상 위의 텀블러를 양손으로 감싸 들고 마시는 모습,
눈은 카메라(웹캠)를 보면서 빨대로 음료를 마시는 중,
살짝 눈웃음, 편안한 분위기
```

### 변형 3 — 거울 보며 준비하는 모습 (End Frame용)

```
이 여성이 모든 것 그대로 유지,
카메라 뒤쪽 벽에 거울이 있다고 가정,
거울을 보며 흰색 가디건을 걸치고 옷매무새를 정리하는 모습,
한 손으로 가디건 단추를 잡고 다른 손으로 머리를 넘기는 동작,
자연스러운 외출 준비 느낌
```
