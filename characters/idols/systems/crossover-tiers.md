# Crossover System

> 이 문서는 크로스오버 시스템의 규칙과 현재 상태를 관리합니다.

---

## 크로스오버 등급

| 등급 | 이름 | 설명 | 예시 |
|------|------|------|------|
| **Tier 1** | Easter Egg | MV/앨범에 다른 그룹의 상징이 숨겨짐 | ZENITH MV에 "DUALITY" 문구 (ECLIPSE 연결) |
| **Tier 2** | Cameo | 다른 그룹 멤버가 MV에 잠깐 등장 | SAKURA MV에 PRISM 멤버 뒷모습 |
| **Tier 3** | Collaboration | 정식 콜라보 싱글/무대 | DAWN BRIDGE 합동 콘서트 (STARFALL x AKATSUKI) |
| **Tier 4** | Arc Crossover | 세계관 스토리가 정식으로 교차 | ZENITH x HORIZON 크로스오버 예고 |
| **Tier 5** | Convergence Event | 전체 유니버스 합류 이벤트 | (미래, 미공개) |

## 크로스오버 추가 규칙

1. 같은 소속사 내 크로스오버는 자유 (선시너지 보장)
2. 다른 소속사 간 크로스오버는 Resonance Grid의 Link 타입에 따라 전개
3. Tier 4 이상은 반드시 양측 그룹 유니버스 파일에 기록
4. Tier 5는 전체 유니버스 이벤트로, 이 문서에서 총괄 관리
