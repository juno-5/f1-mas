# Resonance Grid

> 이 문서는 Resonance Grid의 규칙과 현재 상태를 정의합니다.

---

## 개요

모든 그룹/소속사는 **Resonance Grid** 위의 노드(node)다. 노드 사이에는 **연결선(Link)**이 존재하며, 연결의 종류에 따라 세계관이 교차한다.

---

## 연결 타입

| 타입 | 코드 | 설명 | 예시 |
|------|------|------|------|
| **Harmonic Link** | `H` | 음악적 조화. 합동 공연, 콜라보. | STARFALL ↔ AKATSUKI (DAWN BRIDGE 콘서트) |
| **Dissonance Link** | `D` | 음악적 충돌. 라이벌, 경쟁. | STARFALL ↔ NOVA (PRISM vs AURORA) |
| **Echo Link** | `E` | 세계관 공유. 같은 Frequency 영역에 존재. | ZENITH ↔ ECLIPSE (탑과 이중성의 접점) |
| **Bridge Link** | `B` | 문화적 다리. 국경/장르를 넘는 연결. | GLOBAL ONE ↔ 전사 (해외 투어 협력) |
| **Legacy Link** | `L` | 1세대 → 2세대 계승. | NOVA7 → ZENITH (STARFALL 내 계승) |
| **Void Link** | `V` | 위기/갈등. The Void에 가까워지는 연결. | 해산 그룹들의 잔향 |

## 확장 규칙

새 노드(소속사/그룹)를 추가할 때:

1. **노드 정의**: 이름, 지역, Frequency 영역
2. **최소 2개 Link 설정**: 기존 노드와 최소 2개의 연결을 만든다
3. **개별 유니버스 파일 생성**: `{region}/groups/{GROUP}-universe.md`
4. **이 문서의 Grid에 추가**: 위 다이어그램과 하단 매핑 테이블 업데이트
