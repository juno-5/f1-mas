# STARVERSE Event Protocol

> 사건(Event) 타입의 표준 정의. 각 타입마다 조건/표면 서사/Frequency 진실/상태 변화를 표준화합니다.

---

## Event Types

### 1. Comeback (컴백)

| 항목 | 정의 |
|------|------|
| **코드** | EVT.COMEBACK |
| **트리거** | 소속사 결정 + 시즌 전략 |
| **표면 서사** | 새 앨범/싱글 발매. 팬덤 축제. |
| **Frequency 진실** | 그룹의 Frequency 영역에 새 파동 발생. 세계관 전진. |
| **상태 변화** | R ↑, S ±, Vₚ ± (세부는 update-rules.md 참조) |
| **Memory 생성** | Canon A 노트 (결과 확정 후) |

### 2. Concert (콘서트/투어)

| 항목 | 정의 |
|------|------|
| **코드** | EVT.CONCERT |
| **트리거** | R ≥ 50 + 팬덤 Ht ≥ 50 |
| **표면 서사** | 라이브 공연. 팬과의 만남. |
| **Frequency 진실** | 대규모 팬덤 동기화 → Resonance Event 발동 가능 |
| **상태 변화** | R ↑, S ↑, Vₚ ↓, Fandom Sy/Ht ↑ |

### 3. Collab (콜라보)

| 항목 | 정의 |
|------|------|
| **코드** | EVT.COLLAB |
| **트리거** | H Link LS ≥ 40 + 소속사 승인 |
| **표면 서사** | 합동 싱글/무대. "놀라운 만남!" |
| **Frequency 진실** | 두 Frequency 영역의 공명 → Link 강화 |
| **상태 변화** | R ↑(양측), LS ↑, Rk ↓ |

### 4. RivalArc (라이벌 아크)

| 항목 | 정의 |
|------|------|
| **코드** | EVT.RIVALARC |
| **트리거** | D Link LS ≥ 40 + 팬덤 Heat 상호 ≥ 60 |
| **표면 서사** | "음원 차트 경쟁", "연말 대상 라이벌" |
| **Frequency 진실** | Dissonance가 양쪽 공명을 증폭하지만 안정성을 침식 |
| **상태 변화** | R ↑↑, S ↓, Vₚ ↑ |

### 5. Disband (해산/졸업)

| 항목 | 정의 |
|------|------|
| **코드** | EVT.DISBAND |
| **트리거** | Vₚ ≥ 90 또는 소속사 결정 |
| **표면 서사** | "계약 만료", "멤버 사정", "새 출발" |
| **Frequency 진실** | Void가 해당 영역을 삼킴. 잔향(V Link)이 남아 주변에 영향 |
| **상태 변화** | 해당 그룹 → 소멸. 주변 노드 Vₚ ↑ |

### 6. VoidLeak (Void 누출)

| 항목 | 정의 |
|------|------|
| **코드** | EVT.VOIDLEAK |
| **트리거** | 그룹 Vₚ ≥ 70 또는 V Link 잔향 누적 |
| **표면 서사** | "소속사 위기설", "활동 중단 루머", "팬덤 분열" |
| **Frequency 진실** | Void가 Frequency 영역을 침식. 공명이 약해짐. |
| **상태 변화** | 주변 노드 R ↓, S ↓, Vₚ ↑ |
| **대응** | 공연/콜라보/팬덤 동기화로 공명 회복 필요 |

### 7. BridgeTour (브릿지 투어)

| 항목 | 정의 |
|------|------|
| **코드** | EVT.BRIDGETOUR |
| **트리거** | B Link LS ≥ 50 |
| **표면 서사** | "해외 투어", "글로벌 진출" |
| **Frequency 진실** | 새 지역의 Frequency 영역과 접촉. Grid 확장. |
| **상태 변화** | R ↑, B Link LS ↑, 새 지역 노드와 연결 |

### 8. ResonanceEvent (공명 이벤트)

| 항목 | 정의 |
|------|------|
| **코드** | EVT.RESONANCE |
| **트리거** | R ≥ 80 + Fandom Sy ≥ 80 + Ht ≥ 70 |
| **표면 서사** | "역대급 콘서트", "기록 갱신", "전설적 무대" |
| **Frequency 진실** | 물리적으로 설명 불가능한 에너지 폭발. Frequency 영역 확장. |
| **상태 변화** | R +15, S -10 (불안정), Vₚ -(공명으로 Void 후퇴), 새 Link 생성 가능 |

### 9. ConvergenceOmen (수렴 전조)

| 항목 | 정의 |
|------|------|
| **코드** | EVT.CONVERGENCE_OMEN |
| **트리거** | 3개 이상 그룹 동시 ResonanceEvent 또는 특수 조건 |
| **표면 서사** | "K-pop 역사의 전환점", "세계적 현상" |
| **Frequency 진실** | The Convergence의 전조. 모든 Frequency가 미세하게 공진. |
| **상태 변화** | 모든 노드 R +3, 모든 V Link Rk +5 |
