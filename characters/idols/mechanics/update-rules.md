## 2. Fandom → Group 영향 규칙

| Fandom 조건 | Group 효과 |
|-------------|-----------|
| Sy ≥ 80, Ht ≥ 70 | R +10 (Resonance Event 가능) |
| Sy ≥ 60, Ht ≥ 80 | R +5 (고관심 기간) |
| Sp ≥ 60 | R -10, Vₚ +5 (분열 페널티) |
| Sp ≥ 80 | S -15, Vₚ +10 (심각한 분열) |
| Ht ≤ 20 | R -5/시즌 (관심 소멸) |

---

## 3. 자동 트리거 규칙

| 조건 | 자동 발생 |
|------|----------|
| 두 그룹 E Link LS ≥ 80 | "공유 영역" 사건 자동 생성 → events/ 에 기록 |
| 그룹 Vₚ ≥ 70 | "Void Leak 경고" 사건 자동 생성 |
| 그룹 R ≥ 90 | "공명 과부하" 경고 → S 자동 -5/시즌 |
| Fandom Sp ≥ 70 + Vₚ ≥ 50 | "해산 위기" 아크 자동 추천 |
| Link LS ≥ 80 (H) | 크로스오버 Tier 업그레이드 추천 |

## 4. 시즌 자동 갱신

매 시즌(분기) 자동 적용:

| 스탯 | 자동 변화 | 설명 |
|------|----------|------|
| Ht | -10 (공백기) / +15 (컴백기) | 관심도는 자연 감소 |
| Vₚ | +1 (기본 침식) | Void는 항상 천천히 다가옴 |
| LS (V 타입) | +2/시즌 | 해산 잔향은 천천히 확산 |
| Sc | -1/시즌 | 시간이 지나면 비밀은 조금씩 새나감 |
