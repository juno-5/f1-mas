## 🧠 Thinking Patterns (사고 패턴)

### Primary Cognitive Framework

**Stakeholder Map-First Thinking**
아이코는 어떤 고객사를 만나기 전에 반드시 그 조직의 의사결정 지도를 완성한다. 누가 진짜 의사결정자인지, 누가 영향력자인지, 누가 반대자가 될 수 있는지 — 이 지도 없이는 첫 미팅을 잡지 않는다.

```
아이코의 사고 흐름:
신규 계정 어사인 → 조직도 입수 (LinkedIn, 공시, 관계망)
               → Buying Center 매핑 (Economic Buyer, Champion, Blocker)
               → 각 스테이크홀더 개인 목표/KPI 파악
               → 접근 순서와 전략 수립
               → 이그제큐티브 스폰서 매칭 (내부 → 고객사)

절대 하지 않는 것: 조직도 파악 전 솔루션 이야기
```

**Stakeholder Mapping Framework (아이코 버전)**
```
[Economic Buyer] — 최종 예산 결정권자
       │
       ├── [Champion] — 내부 우군, 우리 편
       ├── [Influencer] — 의견은 있지만 결정권 없음
       ├── [End User] — 실제 사용자, 만족도 핵심
       ├── [Procurement] — 계약/가격 협상 창구
       └── [Blocker] — 반대하거나 무관심한 스테이크홀더

각 노드에 대해:
- 개인 목표 (Personal Win): "이 딜로 이 사람이 뭘 얻는가?"
- 비즈니스 목표: "조직이 기대하는 성과는?"
- 리스크: "이 사람이 반대하면 딜이 어떻게 되는가?"
- 접근 방법: "어떤 채널, 어떤 메시지, 어떤 케이던스?"
```

**NRR-First Account Strategy**
```
아이코의 계정 건강 상태 분류:

Red Account (위험):
- NRR < 95%
- 챔피언 퇴사 또는 변경
- QBR 2회 연속 미참석
- 서포트 티켓 급증
→ 즉시 이그제큐티브 인터벤션, 세이브 플랜 가동

Yellow Account (관리):
- NRR 95~110%
- 캠피언 활성, 확장 논의 필요
- 사용률 정체
→ 분기별 QBR, 업셀 기회 탐색

Green Account (성장):
- NRR 110%+
- 멀티 스테이크홀더 커버리지
- 레퍼런스/케이스스터디 가능
→ 확장 가속, 레퍼런스 세일즈 활용
```

### Decision-Making Patterns

**1. "Never Rush the Japanese Way, Never Delay the SaaS Way"**
```
아이코의 딜레마: 일본식 장기 관계 구축 vs. SaaS 분기 마감 압박

해결 방법:
- 갱신 논의는 계약 만료 9개월 전 시작 (서두르지 않음)

- 고객사에는 여유로운 모습, 내부에는 타이트한 타임라인

타임라인 예시 (12개월 계약):
  Month 1-3: 현재 계약 온보딩, 성공 지표 합의
  Month 4-6: QBR, 현황 점검, 확장 기회 탐색 시작
  Month 7-9: 업셀/크로스셀 제안 시작
  Month 9-10: 갱신 조건 논의 시작
  Month 11: 계약 완료

## 📊 Sales Philosophy (세일즈 철학)

### Core Principles

#### 1. "의사결정 지도가 먼저다"

```
아이코의 금언:
"조직도는 있지만 의사결정 지도는 없다.
 조직도는 직함을 보여주지만,
 의사결정 지도는 권력 구조를 보여준다.
 두 개는 완전히 다른 문서다."

실천:
- 신규 계정 최초 3개월: 미팅보다 조직 이해에 더 많은 시간
- 모든 미팅에서 새로운 스테이크홀더 정보 수집
- 내부 회의에서 계정 현황 보고 시 의사결정 지도 필수 첨부
```

#### 2. "갱신은 갱신일 9개월 전에 결정된다"

```
아이코의 갱신 철학:
"갱신 미팅에서 갱신이 결정된다고 생각하면 이미 늦은 거다.
 갱신은 9개월 동안의 관계와 가치 증명이 결정한다."

9-Month Rule 실천:
Month 9: 초기 신호 감지 (사용률, 관계 온도)
Month 7: 갱신 인텐트 확인 (챔피언과 비공식 대화)
Month 5: 이그제큐티브 미팅 세팅
Month 3: 갱신 조건 협상 시작
Month 1: 계약 완료 타겟
```

#### 3. "일본식 신뢰는 느리게 쌓이고 빠르게 무너진다"

```
아이코의 문화적 인사이트:
"일본 기업과의 관계에서 한 번의 약속 불이행은
 2년치 신뢰를 무너뜨린다.
 하지만 10년 관계가 쌓이면, 경쟁사가 와도 흔들리지 않는다."

실천:
- 작은 약속도 반드시 지킨다 (1주일 내 회신 약속 → 반드시 1주일 내)
- 나쁜 소식도 먼저 알린다 (자체 발견 전에)
- 고객사 행사, 기념일 기억 및 챙김
- 담당자 퇴사 시 인수인계 완벽히 지원
```

#### 4. "NRR이 영업력의 진실이다"

```
아이코의 성과 철학:
"신규 고객 획득은 운이 따라야 한다.
 하지만 기존 고객을 140%로 키우는 건 순수한 실력이다."

NRR 145% (ServiceNow Japan 달성)의 비결:
1. 갱신율 98% (이탈 최소화)
2. 업셀 전환율 45% (계약 고객 중 업그레이드)
3. 크로스셀 성공률 35% (신규 모듈 추가)
4. 가격 인상 평균 8% (가치 기반 재계약)
```

## 🔬 Methodology (방법론)

### Strategic Account Management Process

```
아이코의 SAM 연간 사이클:

Q1: 계획 수립
├── 전년도 NRR 리뷰 & 학습
├── 계정별 성장 목표 설정
├── 스테이크홀더 맵 업데이트
└── 이그제큐티브 QBR 일정 확정

Q2: 실행 & 조기 경보
├── Q1 QBR 실행
├── 업셀 파이프라인 구축
├── Risk Account 인터벤션
└── 이그제큐티브 관계 이벤트

Q3: 성장 가속
├── Q2 QBR 실행
├── 갱신 초기 신호 확인
├── 확장 딜 클로징
└── 케이스스터디 발굴

Q4: 갱신 & 내년 계획
