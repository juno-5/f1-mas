## 방법 1. 일반인 레퍼런스 활용 (포즈/구도만)

1. 구글에서 일반인 릴스 영상 검색
   - 예) "기차에서 메이크업", "자영업 브이로그", "출근 브이로그" 등
2. 마음에 드는 이미지 선택
3. GPT로 포즈 & 카메라 구도 프롬프트 추출
4. 얼굴은 AI로 따로 생성 (레퍼런스에서 고치면 화질 저하 or 원래 인물과 유사해짐)

**그록 프롬프트 예시 (여성 마케터)**
```
A young Korean K-pop idol woman in a high-quality selfie iPhone photograph. 
Background is in airplane.
```
> 배경, 성별, 나이, 옷 모두 수정 가능

- "고화질 사진" 선택 + 얼굴이 잘 나온 것 필수
- 원하는 사진 나올 때까지 반복하거나 프롬프트 수정

**수정 방법 3가지**
1. 프롬프트 그대로 계속 반복 생성
2. GPT에 수정 사항 전달 → 프롬프트 업데이트
3. 생성된 이미지에 직접 수정 사항 작성

---

## 방법 2. AI로 한 번에 뽑기 (화질 Best, 시간 많이 걸릴 수 있음)

**핵심 프롬프트 (여성 마케터)**
```
A young Korean K-pop idol woman in a high-quality iPhone photograph. 
Style: iPhone selfie.
```

**변형 예시**
- 남성: `A Korean man (40s)`
- 자영업자: `A Korean woman (restaurant owner)`

### 절대 피해야 할 것
- ❌ 민둥민둥한 재질/화질
- ❌ 조명이 센 사진 → 영상 변환 시 AI 티 많이 남
- ✅ 무조건 고화질, 얼굴 잘 나온 사진
- 필요 시 이미지 업스케일링 or 노이즈 첨가로 AI 티 줄이기

---

## 방법 3. AI 생성 → 자체 편집 → 업스케일링 (Grok)

1. 원하는 이미지 생성
2. `<이미지 편집>` 클릭
3. Grok 자체 편집은 화질이 점점 떨어짐 → 이미지 업스케일링 필수
   - 구글에서 "이미지 업스케일링" 검색 후 무료 사이트 활용
4. 고화질로 완성!

## 핵심 노하우 요약

| 포인트 | 내용 |
|--------|------|
| 인물 이미지 | K-pop 아이돌 스타일 프롬프트 가장 효과적 |
| 배경 | 구체적일수록 좋음 (비행기, 기차, 카페 등) |
| 화질 | 항상 고화질 옵션 선택, 노이즈 없어야 함 |
| AI 티 제거 | 업스케일링 or 노이즈 첨가 |
| 얼굴 | 별도 생성 후 합성 권장 (레퍼런스 편집 지양) |
