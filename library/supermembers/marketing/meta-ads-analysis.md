## 6. 소재 네이밍 구조 분석

슈퍼멤버스 광고 네이밍 컨벤션:
```
[날짜]-[유형1]-[유형2]-[내용]-v[버전]-LNK-[포맷]

예: 251117-painpoint-vp-광고비줄이고매출올린비밀-v1-LNK-VID
```

**유형 태그:**
- `testimonial` = 고객 후기 / 인플루언서 증언
- `painpoint` = 문제 제기 (불편함, 두려움 자극)
- `vp` = 가치 제안 (혜택, 솔루션 제시)
- `ai` = AI 관련 컨셉 포함
- `branded` = 인플루언서 브랜디드 콘텐츠 (머니멘터리, 맥형, 깡대표 등)

## 7. 핵심 인사이트

### ✅ 잘 작동하는 것들

**1. 브랜디드 + 실시간 성과 = 최저 CPL**
- 머니멘터리 협업 소재가 CPL 5,759원 달성 (전체 최저)
- 인플루언서 신뢰도 + 실제 성과 수치 결합이 전환률 극대화

**2. 30초 짧은 후기 = 고CTR**
- `상위노출-30초` (9.42%), `월매출5000-30초` (8.60%)
- 짧고 핵심만 담은 testimonial이 클릭률 압도적으로 높음

**3. 역발상 페인포인트 = 클릭 유발**
- `마케팅업체속지말기` (10.27%), `무료체험단하지마세요` — 금지형 문구가 궁금증 유발
- `광고비줄이고매출올린비밀` — 비밀/역설 구조가 클릭 유도

**4. AI + 페인포인트 조합 = 예산 효율**
- 30일간상위노출실험 CPL 37,641원 (상위 4위)
- `돈안날리는상위노출` — AI 컨셉 + 손해 방지 메시지

**5. `광고비줄이고매출올린비밀` = 규모 확장성 검증**
- 8.3M원 집행에도 CTR 2.89% 유지, 218건 리드
- 롤오버 없이 대규모 예산 소화 → 메인 소재로 검증됨

### ❌ 개선 필요한 것들

**1. 높은 CTR이 리드로 연결 안 됨**
- `상위노출-30초` CTR 9.42%인데 리드 0건 → 랜딩페이지 문제 가능성
- `매출상승-실사용후기` CTR 8.27%, 리드 0건

**2. 이미지(IMG) 소재 성과 저조**
- `VP-모집X` (이미지): CTR 1.80%, CPC 2,601원
- 전반적으로 VID > IMG 압도적

**3. 일부 페인포인트 소재 CTR 낮음**
- `옆집마케팅비밀` CTR 2.00%, CPL 64,502원 (최고가)
- `맥형_Branded_VP-v1` CTR 1.32% — 브랜디드인데 성과 저조

## 8. 개선된 소재 제작 방향 (권장)

### 🎯 검증된 공식 + 신규 각도 결합

**공식 A: 브랜디드 + 30초 + 수치**
```
구조: [인플루언서 등장] → [충격 수치] → [30초 내 결론]
각도: "저 이거로 월매출 500만 올렸어요" (30초)
참고 소재: 머니멘터리_자동화마케팅 (CTR 12.77%)
```

**공식 B: 역발상 금지형 훅 + 페인포인트**
```
구조: [하지 말라는 것] → [왜?] → [대안 제시]
각도: "마케팅 업체 절대 믿지 마세요" / "체험단 그만 하세요"
참고 소재: 마케팅업체속지말기 (CTR 10.27%)
```

**공식 C: AI + 실험 + 증거**
```
구조: [AI로 실험] → [30일 결과] → [비용 절감 수치]
각도: "AI로 30일간 상위노출 실험했더니..."
참고 소재: 30일간상위노출실험 (CTR 3.87%, CPL 37,641원)
```

**공식 D: 공포 → 안도 구조**
```
구조: [불경기/폐업 위기] → [슈퍼멤버스 발견] → [매출 회복]
각도: "마케팅 비용 다 날리다가..."
참고 소재: 광고비줄이고매출올린비밀 (8.3M 집행, 안정적)
```
