# 슈퍼멤버스 서비스 이해

> 출처: Notion (mayacrew 워크스페이스) + Meta 광고 데이터 분석  
> 정리일: 2026-02-22

---

## 1. 서비스 본질

**슈퍼멤버스는 "진짜 손님 리뷰 자동화" 플랫폼이다.**

자영업자(사장님)와 블로거(리뷰어)를 연결해주는 양방향 마켓플레이스.

```
사장님 (구독) ←→ 슈퍼멤버스 플랫폼 ←→ 블로거 (방문 + 리뷰)
```

**왜 필요한가?**  
한국 자영업자의 신규 손님 유입 경로:
```
네이버 플레이스 검색 → 순위 상위 매장 클릭 → 리뷰 확인 → 방문
```
네이버 플레이스 순위는 **진짜 손님의 리뷰 수와 품질**로 결정된다.  
슈퍼멤버스는 이 리뷰를 자동으로 만들어주는 서비스.

---

## 2. 작동 메커니즘

### 사장님(업체) 입장

1. 슈퍼멤버스 구독 (월 정기 결제)
2. 매장 정보 등록
3. 슈퍼멤버스가 자동으로 등급에 맞는 블로거 매칭
4. 블로거가 매장 방문 → 가맹점 확인 코드 입력 (현재 **코드: 55**)
5. 블로거가 네이버 블로그에 리뷰 작성
6. 리뷰 누적 → 네이버 플레이스 순위 상승 → 신규 손님 유입

> ⚠️ 직원/알바에게도 코드(55) 공유 필수 — 모르면 블로거가 리뷰 못 씀

### 블로거 입장

1. 슈퍼멤버스 앱 가입
2. 등급 심사 통과 (등급별 할인율 달라짐)
3. 매장 방문 예약 → 이용 후 확인 코드 입력
4. 블로그 리뷰 작성 (할인/무료 혜택 제공)

---

## 3. 블로거 등급 시스템

| 등급 | 기준 | 심사 방법 |
|------|------|---------|
| 다이아 | 순위 + 방문자수 + 원고 품질 | 수동 심사 |
| 블랙 | 순위 + 방문자수 + 원고 품질 | 수동 심사 |
| 그 외 | 순위 기준치 | 자동 심사 |

**원고 품질 기준 (다이아/블랙)**:
- 글자수 공백 제외 1,300자 이상
- 사진 13장 이상
- 사진 + 글 비율 균형 (10줄 단위)
- 전체 가독성

**등급 재심사**: 매월 말

---

## 4. 상품 구조

- **플랜**: plan 값에 따라 기간 결정 (plan × 60일 = 최대 기간)
- **결제 방식**: 정기 결제 (최근 3차수 데이터 기반)
- **연장 로직**: 블로거 모집 미달 시 자동 기간 연장
- **유형**: 프리미엄 체험단(A) / 제품 일반 체험단

## 5. 자영업자가 슈퍼멤버스를 쓰는 이유

### 핵심 고통

| 상황 | 감정 |
|------|------|
| 마케팅 업체에 돈 날린 경험 | 배신감, 분노 |
| 네이버 검색 안 됨 | 무력감, 답답함 |
| 매출 줄어드는 매달 | 공포, 폐업 압박 |
| 혼자 SNS/블로그 운영 | 탈진, 시간 부족 |

### 슈퍼멤버스가 해결하는 것

```
기존 대안                    슈퍼멤버스
────────────────────────────────────────
마케팅 업체: 150~300만원/월  → 15만원/월
가짜 리뷰: 플랫폼 제재 위험  → 진짜 손님 리뷰 (안전)
무료 체험단: 신뢰도 낮음      → 등급 심사된 블로거
직접 SNS 운영: 시간 없음      → 완전 자동화
```

### 선택 이유 (광고 분석 기반)

1. **자동화** — "떠먹여드린다" = 알아서 해준다
2. **진짜 리뷰** — 가짜 아닌 실방문 손님
3. **낮은 진입장벽** — 월 15만원, 카드 등록 없음
4. **검증된 효과** — 상위노출, 예약 증가 실사례
5. **시간 절약** — 장사에만 집중 가능

## 6. 경쟁사 포지셔닝

경쟁사 (Notion 분석 문서 기준):
- **레뷰 코퍼레이션**
- **리뷰 노트**
- **블덱스**

슈퍼멤버스 차별점:
- 블로거 등급 심사 시스템 → 리뷰 품질 관리
- 자동 매칭 시스템 → 사장님 개입 최소화
- 진짜 방문 확인 (코드 시스템) → 플랫폼 신뢰도

## 7. 현재 비즈니스 과제

### 해지율 (Churn)

- 현재 해지 방어율: **4~5%** (156건 중 5건 방어)
- 해지 유형:
  - **우발적 해지**: 일시적 불만, 개입 시 방어 가능
  - **이상적 해지**: 장기 고민 후 결정, 방어 어려움

### 해지 원인 가설 (사장님 VOC 기반)

- 기대와 실제 효과의 차이 (상위노출 기대치 vs 실제 속도)
- 가격 대비 체감 성과 부족
- 블로거 모집 지연

### 개입 가능 지점

해지 요청 **이전** 사전 개입이 핵심:
- 리뷰 증가 추이 실시간 알림
- 순위 변화 보고서
- 성과 미달 시 CS 먼저 연락

## 8. 콘텐츠/광고 전략 인사이트

(→ `ads/` 폴더 참조)

**검증된 광고 공식:**

| 공식 | 예시 | CTR |
|------|------|-----|
| 브랜디드 + 수치 | 머니멘터리 협업 | 12.77% |
| 역발상 금지형 훅 | "마케팅업체 속지마세요" | 10.27% |
| 30초 압축 후기 | "상위노출 30초" | 9.42% |
| 폐업 위기 서사 | "폐업신고서 뽑았습니다" | 예상 10%+ |

**핵심 타겟 언어:**
- "마케팅비", "상위노출", "손님이 없다", "떠먹여드린다"
- "월 15만원", "자동으로", "진짜 손님", "리뷰"
