# 슈퍼멤버스 광고 소재 자동 베리에이션 워크플로우

> 목표: 기존 상위 소재 구조 유지 + 배경/인물/대사만 교체 → 자동 제작  
> 1차 목표: 상위 소재와 동일 퀄리티 소재를 자동으로 찍어내는 파이프라인  
> 작성일: 2026-02-22

---

## 핵심 전략: "검증된 틀은 건드리지 않는다"

```
기존 상위 소재 = 검증된 공식
              ↓
배경 / 인물 / 숫자 / 키워드만 교체
              ↓
구조는 100% 동일하게 유지
              ↓
A/B 테스트로 최적 베리에이션 선별
```

## 베리에이션 템플릿 A — "머니멘터리_자동화마케팅" 계열
> 원본 CTR: 12.77% | 구조: 브랜디드 + 현장 인물 + "떠먹여드립니다" 카피

### 원본 구조 분해
```
[0~2초]  훅: 현장 인물 + 대형 핵심 카피 (검정 상단 바)
[2~12초] 빌드: 인물 인터뷰 + 체험담 (자동화/편리함 강조)
[12~18초] 솔루션: 앱 화면 + 수치
[18~20초] CTA: 추천인 코드 + 행동 유도
```

### 베리에이션 변수 (바꿀 것)

| 변수 | 원본 | 베리에이션 1 | 베리에이션 2 | 베리에이션 3 |
|------|------|------------|------------|------------|
| **인플루언서/인물** | 머니멘터리 (식당) | 맥형 (카페) | 깡대표 (뷰티샵) | 일반 자영업자 (네일샵) |
| **배경** | 식당 내부 | 카페 | 뷰티샵 | 네일샵 |
| **핵심 카피** | "떠먹여드립니다" | "알아서 다 해줘요" | "신경 쓸 게 없어요" | "저 대신 해줘요" |
| **추천인 코드** | 머니멘터리 | 맥형 | 깡대표 | 없음 (직접 CTA) |
| **업종 키워드** | 매장 | 카페 | 뷰티샵 | 네일샵 |

### 베리에이션 1 — 카페 사장님 버전 스크립트
```
[0~2초]  
화면: 카페 카운터 뒤 사장님 (30대 여성, 바리스타 앞치마)
텍스트 상단 바: "마케팅 알아서 다 해줘요"
사운드: 커피 그라인딩 소리 → 뚝

[2~12초]
대사: "저 원래 인스타 마케팅, 블로그 관리 혼자 다 했거든요.
       근데 커피 만드는 것도 바쁜데 마케팅까지 하니까
       둘 다 엉망이 되더라고요.
       슈퍼멤버스 쓰고 나서는 마케팅은 아예 신경 안 써요."

[12~18초]
화면: 슈퍼멤버스 앱 리뷰 알림 쏟아지는 화면
대사: "등급별 블로거가 자동으로 매칭되고
       방문까지 알아서 되니까요.
       저는 그냥 커피만 잘 만들면 돼요."

[18~20초]
텍스트 하단 바: "추천인: [인플루언서명]"

## 자동화 파이프라인 (노드 연결 후 즉시 실행)

```
STEP 1: 베리에이션 변수 세팅 (5분)
┌─────────────────────────────────┐
│ 어떤 템플릿? A / B / C          │
│ 인물 업종? 카페/식당/뷰티/네일  │
│ 핵심 수치? (직접 입력)          │
│ 훅 키워드? (선택 or 자동 생성) │
└─────────────────────────────────┘
         ↓

STEP 2: 인물 이미지 생성 (15~20분)
┌─────────────────────────────────┐
│ Nano Banana Pro / Grok         │
│ 업종별 프롬프트 자동 선택       │
│ 3~5장 생성 → 최적본 선택        │
│ 배경: 해당 업종 매장 내부        │
└─────────────────────────────────┘
         ↓

STEP 3: 영상 생성 (20~30분)
┌─────────────────────────────────┐
│ Google Flow / Kling AI         │
│ 이미지 → Start Frame           │
│ 스크립트 기반 JSON 프롬프트 자동 생성 │
│ 씬별 영상 생성 + 루프백 연결    │
└─────────────────────────────────┘
         ↓

STEP 4: 후반 작업 (10분)
┌─────────────────────────────────┐
│ 텍스트 오버레이 자동 삽입        │
│ (원본과 동일한 폰트/위치/색상)  │
│ BGM + 사운드 효과               │
│ Topaz 4K 업스케일              │
└─────────────────────────────────┘
         ↓

STEP 5: Meta 광고 업로드 (5분)
┌─────────────────────────────────┐
│ Meta API 자동 업로드            │
│ 기존 상위 캠페인에 추가         │
│ 네이밍: 260222-[유형]-[컨셉]-v1-LNK-VID │
│ 초기 예산: 10만원/일           │
└─────────────────────────────────┘
         ↓

STEP 6: 성과 모니터링 (자동)
┌─────────────────────────────────┐
│ Meta API로 48시간 후 CTR 확인  │
│ CTR > 5%: 예산 확대            │
│ CTR < 2%: 중단 + 다음 베리에이션 │
└─────────────────────────────────┘
```

## 우선순위 베리에이션 실행 순서 (노드 연결 당일)

### 1회차 (당일)
```
□ 템플릿 A-1: 카페 사장님 + "알아서 다 해줘요"
□ 템플릿 B-1: "무료 체험단 그만 하세요"
□ 템플릿 C-1: "3일 만에 1위"
```

### 2회차 (48시간 후 성과 확인 후)
```
□ 성과 좋은 템플릿 → 업종 추가 (뷰티샵, 네일샵, 헬스장)
□ 성과 나쁜 템플릿 → 훅 카피 변경 후 재제작
```

### 월간 목표
```
주 1회 제작 사이클 (월 4~6개 신규 소재)
→ 항상 3~5개 A/B 테스트 소재 동시 운영
→ 상위 소재는 예산 자동 확대
→ 하위 소재는 자동 중단
```

## 텍스트 오버레이 스타일 가이드 (원본 소재 동일 적용)

```

하단 바:
  배경색: #000000 (검정)  
  주요 텍스트: #FFFFFF, Bold, 40pt
  강조 텍스트: #FFD700 (노란색), Bold, 44pt
  위치: 화면 하단 12~15% 영역

중간 자막:
  텍스트: #FFFFFF, 그림자 효과, 32pt

수치 강조:
  색상: #FF4444 (빨간색) 또는 #FFD700 (노란색)
  크기: 주변 텍스트 대비 1.3~1.5배
