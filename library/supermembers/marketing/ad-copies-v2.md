## 핵심 소구 포인트 3가지 (세일즈 스크립트 기반)

```
1. 퀄리티 — "상위 블로거만 자연스럽게 남는 구조"
   (등급 차등 혜택 → 낮은 블로거는 사비 → 좋은 블로거만 옴)

2. 자동화 — "사장님은 등록만 하면 됨"
   (배민 모델 — 블로거가 알아서 찾아옴)

3. 가성비 — "인당 1.5만원, 경쟁사 5만원"
   (월 15만원, 최소 8명 보장, 안 되면 무료 연장)
```

---

## 카피 세트 A — 퀄리티 소구

### A-1. 훅 카피 (5~8단어)
```
"돈 내고 온 손님이 쓴 리뷰는 다릅니다"
"상위 블로거가 스스로 찾아오는 이유"
"무료 체험단은 안 씁니다. 이유 있어요"
"영향력 없는 블로거 리뷰, 오히려 독 됩니다"
```

### A-2. 본문 카피
```
기존 체험단은 이렇게 돌아가요.
영향력이 낮아도 무조건 무료 제공.
그 사람이 쓴 리뷰... 손님들 믿을까요?

슈퍼멤버스는 달라요.
다이아 등급 블로거 = 100% 혜택
낮은 등급 = 본인이 일부 부담.

결과적으로 영향력 높은 블로거만 남아요.
자연스럽게.
```

### A-3. CTA
```
"퀄리티 다른 리뷰, 무료로 경험해보세요"
"상위 블로거 자동 매칭 → 지금 시작"
```

## 카피 세트 B — 자동화 소구

### B-1. 훅 카피
```
"블로거가 알아서 찾아옵니다"
"등록 하나로 마케팅 끝납니다"
"배달의 민족처럼 알아서 옵니다"
"사장님은 맛있는 것만 만드세요"
```

### B-2. 본문 카피
```
기존 체험단:
블로거 모집 → 심사 → 선정 → 제품 배송 → 리뷰 확인
→ 2~3주 걸리고, 사장님이 다 해야 함

슈퍼멤버스:
광고 등록 하나 → 블로거가 알아서 찾아와서 방문 → 리뷰 자동 쌓임
→ 배달의 민족이랑 같은 구조

사장님이 할 일?
영업시간에 방문하는 블로거한테 코드 55 입력해주는 것 뿐.
```

### B-3. CTA
```
"등록 3분, 나머지는 슈퍼멤버스가"
"지금 바로 시작 → 카드 등록 없음"
```

## 카피 세트 C — 가성비/수치 소구

### C-1. 훅 카피
```
"레뷰 다이아 블로거 5명 = 25만원"
"같은 블로거, 우리는 10명에 15만원"
"월 15만원으로 상위 블로거 10명 이상"
"마케팅 비용 70% 줄인 사장님 실화"
```

### C-2. 본문 카피
```
마케팅 대행사: 월 150~300만원
무료 체험단: 퀄리티 낮고 신뢰 없음
레뷰 프리미엄(다이아 5명): 25만원 → 인당 5만원

슈퍼멤버스:
월 15만원 → 최소 8명 방문
인당 1만5천원

8명 안 되면? 무료 연장입니다.
```

### C-3. CTA
```
"인당 1.5만원, 리뷰 10개 이상 → 지금 시작"
"효과 없으면 무료 연장 → 리스크 제로"
```

## 카피 세트 D — 역발상 / 금지형 훅 (검증된 패턴)

### D-1. 기존 체험단 안티테제
```
훅: "무료 체험단 지금 당장 끊으세요"

본문:
무료로 먹은 사람이 쓴 리뷰.
손님들 다 알아요, 공짜라는 거.
그 리뷰, 네이버도 낮게 봅니다.

슈퍼멤버스 블로거는 돈 내고 와요.
자기 돈 내고 온 사람이 쓴 리뷰가
얼마나 다를지 생각해보세요.
```

### D-2. 마케팅 업체 안티테제
```
훅: "마케팅 업체 전화 다시는 받지 마세요"

본문:
먼저 전화 오는 업체, 이유가 있어요.
사장님 간절함에 돈 버는 거예요.

슈퍼멤버스는 인당 1.5만원입니다.
업계 최저. 검색해보세요.
```

### D-3. SNS 광고 안티테제
```
훅: "인스타 광고비 태우는 거 그만하세요"

본문:
광고 켜면 손님 오고, 끄면 바로 없어요.
그 돈으로 리뷰를 쌓으세요.
리뷰는 광고 꺼도 계속 일합니다.
```

## 카피 세트 E — 감정/서사 (폐업 위기 → 반전)

### E-1. 폐업 위기 서사
```
훅: "이번 달 폐업 신고서 뽑아봤습니다"

본문:
임대료 350.
직원 월급 280.
이번 달 매출 480.
계산해보세요.

근데 슈퍼멤버스 쓰고 한 달 뒤에
예약이 차기 시작했어요.
리뷰가 쌓이니까 네이버에서 찾아오더라고요.

폐업 생각하기 전에 이거 한 번만 해보세요.
```

### E-2. 손님 없는 게 내 잘못 아님
```
훅: "손님이 없는 게 내 잘못이 아니었어요"

본문:
더 맛있게 만들려고 레시피 바꿨어요.
인테리어도 손봤어요.
가격도 낮췄어요.

그래도 손님이 없었는데.

알고 보니 네이버에서 우리 가게가 안 보였던 거예요.
음식이 나쁜 게 아니라 아무도 못 찾은 거예요.

슈퍼멤버스로 리뷰 쌓고 나서 달라졌어요.
```

## 카피 길이별 포맷

### 30초 영상용 (CTR 최적)
```
[0~3초]  훅 1줄 (충격 or 수치 or 금지형)
[3~20초] 페인 공감 + 기존 vs 슈퍼멤버스 구조 설명
[20~27초] 수치 + 결과 (리뷰 몇 개, 순위 몇 위)
[27~30초] CTA (무료 시작 / 카드 없음)
```

### 45초 영상용 (서사형)
```
[0~3초]  훅 (감정 폭발)
[3~15초] 위기 상황 (임대료, 매출, 빚)
[15~30초] 슈퍼멤버스 발견 + 구조 설명
[30~40초] 결과 (순위, 리뷰, 예약)
[40~45초] CTA
```
