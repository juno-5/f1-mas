# 슈퍼멤버스 세일즈 스크립트

> 출처: Notion (mayacrew 워크스페이스)  
> 최신 버전: 25.7.15 수정본 기준  
> 정리일: 2026-02-22

---

## 핵심 세일즈 논리 흐름

```
1. 왜 마케팅이 필요한가?
        ↓
2. 왜 네이버인가?
   → 한국인이 가장 많이 검색하는 포털 = 고객 신뢰의 기반
        ↓
3. 왜 블로그 리뷰인가?
   → 네이버가 "진짜 고객 신호"로 인식 = 플레이스 순위에 직접 영향
        ↓
4. 왜 슈퍼멤버스인가?
   → 영향력 있는 블로거만 자연 선별되는 구조 + 완전 자동화
```

---

## 6단계 통화 스크립트

### 1단계 — 인사 & 콜 목적
```
"안녕하세요 담당자님, 슈퍼멤버스입니다.
저희 측 도입 문의 주셔서 체험단 진행 안내드리려고 연락드렸습니다.
잠시 통화 괜찮으실까요?"
```

**담당자 반응별 대응**
- "네 괜찮아요" → 바로 2단계
- "지금 바쁜데요" → "5분만 드려도 될까요? 핵심만 말씀드릴게요"
- "간단히만 알려주세요" → 3단계 핵심만 압축

---

### 2단계 — Pain 포인트 파악

**체험단 경험 없음:**
> "혹시 기존에 블로그 체험단 진행해보신 적 있으실까요?"
→ 없음: 왜 체험단이 필요한지 먼저 설명 후 서비스 소개

**체험단 경험 있음:**
> "그러시군요! 혹시 어디 체험단 이용해보셨을까요?
> 진행하실 때 가장 아쉬웠던 부분이 어떤 거였나요?"

**대표 페인 포인트 (공감 후 해결책 제시):**
- 영향력 낮은 리뷰어만 신청한다
- 선정 과정이 오래 걸린다 (2~3주)
- 무료 제공 대비 효과가 낮다
- 리뷰 퀄리티 편차가 크다
- 구매평 확보가 어렵다

**Pain 요약 멘트:**
> "말씀하신 부분이 기존 체험단에서 가장 많이 발생하는 문제들이에요.
> 그래서 저희는 이 Pain을 해결하는 방향으로 구조를 만들어놨습니다."

---

### 3단계 — 서비스 설명

**핵심 구조 한 문장:**
> "슈퍼멤버스 체험단은 담당자님 니즈에 맞게
> 모집 방식 1개 + 진행 방식 1개 조합해서 진행 가능합니다."

**기존 체험단 vs 슈퍼멤버스 (실제 멘트):**
> "기존 체험단은 선정된 사람이 영향력이 낮아도 무료로 제품/음식을 제공하고
> 원고료도 지급해야 되는데,
> 슈퍼멤버스는 등급에 따라서 혜택 제공 방법으로 진행하고 있어요.
> 영향력이 높은 블로거분께는 사장님께서 선정하신 금액의 100%,
> 영향력이 낮은 분에게는 일부 할인 혜택으로 진행돼요.
> 영향력 낮은 분들은 사비를 지출해야 해서
> 자연스럽게 퀄리티 높은 블로거들의 리뷰가 많이 쌓이는 구조예요."

**자동화 설명 (실제 멘트):**
> "사장님께서 직접 체험단 알아보시고 확인하고 모집할 필요 없이
> 블로거분들께서 알아서 찾아오시기 때문에 시간 절약이 됩니다.
> 배달의 민족이랑 비슷한데,
> 사장님이 저희를 통해 광고 등록하시면
> 블로거들이 앱 보고 가게 방문 후 리뷰 작성하는 방식이에요."

**추가 소구 (키워드 선정):**
> "블로그 리뷰뿐만 아니라
> 네이버 검색 키워드 순위 확인해보고
> 검색량이 높은 키워드 선정도 도와드리고 있어요."

**가격/후킹:**
> "비용은 한 달에 15만원 (부가세 포함 165,000원)이고,
> 지금은 1+1 프로모션도 있어서 부담 없이 테스트하듯 시작하실 수 있습니다."

**최소 보장:**
> "한 달에 최소 8명이 방문하실 수 있게끔 도와드리고 있고,
> 많이 방문되는 경우 한 달에 3~40명까지 방문하기도 해요.
> 8명이 안 되면 8명 채워질 때까지 무료 연장도 해드리고 있습니다."

---

### 4단계 — 챌린지 대응

| 반박 | 대응 |
|------|------|
| "비싸다" | 레뷰 다이아 5명=25만원(인당 5만원) vs 슈퍼멤버스 10명=15만원(인당 1.5만원) |
| "효과 없을 것 같다" | 1+1 프로모션으로 가볍게 테스트 먼저 |
| "노출이 안 됨" | 키워드 선정 + 리뷰 누적 = 순위 상승 구조 설명 |
| "관리 안 해준다" | 카톡 1:1 영업일 24시간 내 답변 |
| "타 업체랑 뭐가 달라" | 블로거 차등 혜택 구조 + 모집/선정 불필요 |
| "제품가가 부담스럽다" | 대여나 서비스 제공 방식 + 원고료 조합 가능 |

---

### 5단계 — 오퍼 제시

**니즈 기반 조합 정리:**
> "정리드리면, 담당자님 상황에서는
> (선착순/선정형) 방식에 (직접배송/환급금) 조합이 가장 잘 맞습니다."

**패키지 추천:**
> "체험형 패키지로 테스트하듯 시작하시고,
> 반응 보시고 이후에 제품군이 많을 경우 100만원 패키지로 확대하는 흐름으로 많이 진행하세요."

---

### 6단계 — 클로징

**채널 추가:**
> "저희 진행 관련 소통을 카카오톡 채널에서 도와드리고 있습니다.
> 지금 바로 채널 추가만 가능하실까요?"

**다음 단계:**
> "채널 추가 후 오늘 말씀드린 내용 텍스트로 정리해서
> 소개서와 함께 보내드릴게요."

## 고객 유형별 전략 (25.7.15 업데이트)

| 유형 | 핵심 전략 |
|------|---------|
| 체험단 경험 O | 기존과 다른 점 위주 (등급 차등 + 자동화) |
| 체험단 경험 X | 블로그 체험단 필요성 먼저 → 서비스 소개 |
| 당장 진행 희망 | 서비스 + 플랜 안내, 3개월 신규 프로모션 소구 |
| 비용 고민 | 1+1 프로모션 + 1개월 단위 가능 강조 |
| 시작 시기 고민 | 블로거 성수기 강조 / "지금 세팅 도와줄테니 진행하자" |
| 타 업체 비교 | 블로거 영향력 차등 혜택 강조 + 모집/선정 불필요 |

## 웨비나 세일즈 구조 (별도 채널)

웨비나 주제: **"블로거 체험단으로 신규 고객 30% 늘리는 방법"**

슬라이드 흐름:
1. 마케팅이란 무엇인가? (왜 해야 하나)
2. 왜 네이버인가? (가장 많이 검색 = 고객 신뢰)
3. 어떻게 네이버에서 신뢰를 얻는가 (고객 행동 요소)
4. 블로그의 중요성 (영향력 있는 블로거의 필요성)
5. 슈퍼멤버스 소개 (등급제 + 자율성 구조)

목표: 웨비나 1회로 10명 이상 전환
