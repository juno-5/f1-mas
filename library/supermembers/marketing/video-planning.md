## 🎬 기획안 1 — "마케팅 업체 절대 믿지 마세요 시즌2"

**참고 소재**: 마케팅업체속지말기 (CTR 10.27%)  
**유형**: 페인포인트 + 역발상  
**길이**: 30초  
**포맷**: 세로 9:16, VID

### 훅 (0~3초)
```
[화면] 자영업자 표정으로 카메라 응시, 텍스트 오버레이
[대사] "마케팅 업체 절대 믿지 마세요"
[텍스트] "⚠️ 마케팅 업체 믿다가 500만원 날렸습니다"
[사운드] 경고음 or 강한 베이스 히트
```

### 빌드 (3~20초)
```
[화면] 계좌에서 돈 빠져나가는 효과 or 영수증 클로즈업
[대사] "저도 처음엔 유명 마케팅 업체 계약했어요.
        매달 150만원씩, 6개월 썼는데...
        실제로 올라온 리뷰가 가짜인 거 알게 됐어요"
[텍스트 오버레이] "월 150만원 × 6개월 = 900만원"
```

### 피크 (20~27초)
```
[화면] 슈퍼멤버스 대시보드 실시간 화면
[대사] "그래서 직접 찾은 게 슈퍼멤버스예요.
        진짜 손님이 진짜 리뷰를 쓰거든요"
[텍스트] "진짜 손님 × 진짜 리뷰 = 진짜 매출"
```

### 착지/CTA (27~30초)
```
[화면] 로고 + 무료 체험 화면
[대사] "지금 무료로 먼저 써보세요"
[텍스트] "👇 무료 체험 신청하기"
```

**제작 방향:**
- 인물: 40대 남성 자영업자 느낌 (식당/카페 사장님)
- 배경: 매장 내부 or 주방 앞
- 톤: 약간 화난 듯 → 안도로 전환
- 편집: 빠른 컷, 텍스트 강조

## 🎬 기획안 2 — "30일 동안 AI로 상위노출 실험했더니"

**참고 소재**: 30일간상위노출실험 (CTR 3.87%, CPL 37,641원), 돈안날리는상위노출  
**유형**: AI + 실험 + 증거  
**길이**: 45초  
**포맷**: 세로 9:16, VID

### 훅 (0~3초)
```
[화면] 사장님이 스마트폰 화면을 카메라에 보여줌
[대사] "30일 동안 AI 마케팅 실험해봤어요"
[텍스트] "📊 30일 실험 결과 공개"
[사운드] 실험 효과음 or 카운트다운
```

### 빌드 (3~30초)
```
[화면] Day 1 → Day 7 → Day 15 → Day 30 타임랩스 느낌
[대사] "1일차엔 네이버 검색 20위였어요.
        슈퍼멤버스로 진짜 손님 리뷰 모으기 시작했고
        7일차에 12위, 15일차에 5위..."
[텍스트 오버레이] "1일: 20위 → 7일: 12위 → 15일: 5위 → 30일: ?"
```

### 피크 (30~40초)
```
[화면] 네이버 플레이스 1위 화면 클로즈업
[대사] "30일 만에 1위 찍었어요"
[텍스트] "🥇 30일 만에 네이버 1위"
[사운드] 승리/달성 효과음
```

### 착지/CTA (40~45초)
```
[화면] 매장 손님 많아진 비포/애프터
[대사] "손님도 눈에 띄게 늘었고요. 직접 해보세요"
[텍스트] "👇 나도 30일 실험 시작하기"
```

**제작 방향:**
- 인물: 30~40대 자영업자, 밝은 톤
- 배경: 매장 카운터 or 매장 외부
- 핵심: 숫자 변화 그래프/카운터 애니메이션 필수
- 편집: 데이 카운터 효과, 순위 올라가는 애니메이션
