## 기획안 1 — "돈 내고 온 손님 리뷰는 다릅니다" (퀄리티)

**가설**: H1  
**참고 소재**: 머니멘터리_자동화마케팅 (CTR 12.77%)  
**예상 CTR**: 10~13%  
**길이**: 30초

### 스크립트

**[0~3초] 훅**
```
화면: 식당 사장님, 카메라 직시. 진지한 표정.
대사: "무료 체험단 블로거 리뷰... 손님들 믿을까요?"
텍스트: "무료로 먹은 사람 리뷰 vs 돈 내고 온 사람 리뷰"
사운드: 베이스 히트
```

**[3~18초] 설명**
```
대사:
  "기존 체험단은 영향력 없어도 무조건 무료 제공이에요.
   그 사람이 쓴 리뷰, 네이버도 낮게 봐요.
   
   슈퍼멤버스는 달라요.
   다이아 등급 블로거한테는 100% 혜택,
   낮은 등급은 본인이 일부 부담.
   
   그러니까 영향력 높은 블로거만 자연스럽게 남아요."

텍스트 오버레이:
  "다이아 블로거 → 100% 혜택"
  "낮은 블로거 → 사비 부담 → 자연 필터링"
  "결과: 상위 블로거 리뷰만 쌓임"
```

**[18~27초] 결과 + 수치**
```
화면: 슈퍼멤버스 앱 리뷰 화면
대사:
  "월 15만원에 최소 8명.
   경쟁사 다이아 블로거 5명에 25만원이에요.
   인당 1.5만원 vs 5만원."

텍스트:
  "슈퍼멤버스: 인당 1.5만원"
  "경쟁사: 인당 5만원"
```

**[27~30초] CTA**

## 기획안 2 — "블로거가 알아서 찾아옵니다" (자동화 + 배민 비유)

**가설**: H2  
**참고 소재**: 마케팅업체속지말기 (CTR 10.27%)  
**예상 CTR**: 9~12%  
**길이**: 30초

### 스크립트

**[0~3초] 훅**
```
화면: 카페 사장님, 폰 화면 들고 깜짝 놀란 표정
대사: "방금 블로거 방문 알림이 또 왔네요."
텍스트: "제가 모집한 거 아닌데..."
사운드: 카카오톡 알림음 × 3
```

**[3~18초] 자동화 설명**
```
대사:
  "슈퍼멤버스는 배달의 민족이랑 구조가 같아요.
   저는 광고 등록 한 번 했고,
   블로거분들이 앱에서 우리 카페 보고
   알아서 예약하고 오거든요.
   
   제가 모집하고 선정하고 연락하는 거 없어요.
   방문하면 코드 55 입력해주면 끝이에요."

텍스트:
  "1. 광고 등록 1번"
  "2. 블로거가 알아서 방문"
  "3. 리뷰 자동 쌓임"
  "내가 할 일: 코드 55 입력"
```

**[18~27초] 결과**
```
화면: 네이버 플레이스 순위 화면 + 예약 꽉 찬 화면
대사:
  "리뷰 쌓이니까 네이버 순위 올라갔고,
   이번 주 예약이 꽉 찼어요."

텍스트:
  "리뷰 ↑ → 네이버 순위 ↑ → 손님 ↑"
```

**[27~30초] CTA**
```
대사: "월 15만원, 지금 무료로 시작해보세요."
텍스트: "👇 3분 세팅 → 블로거 자동 매칭"
```

## 기획안 3 — "인당 1.5만원 vs 5만원" (수치 대비)

**가설**: H3  
**참고 소재**: 상위노출-30초 (CTR 9.42%)  
**예상 CTR**: 8~11%  
**길이**: 30초

### 스크립트

**[0~3초] 훅**
```
화면: 사장님, 영수증 or 계산 화면 들고 카메라 봄
대사: "레뷰 다이아 블로거 5명에 25만원이에요."
텍스트: "경쟁사: 인당 5만원"
사운드: 계산기 소리
```

**[3~18초] 비교**
```
대사:
  "저도 처음에 거기 썼어요.
   다이아 블로거 5명에 25만원.
   
   슈퍼멤버스는요?
   월 15만원에 최소 8명이에요.
   인당 1만5천원이에요.
   
   그리고 8명 안 되면 무료 연장해줘요.
   리스크가 없어요."

텍스트:
  "경쟁사: 5명 × 5만원 = 25만원"
  "슈퍼멤버스: 8명 × 1.5만원 = 15만원"
  "안 채우면? 무료 연장"
```

**[18~27초] 결과**
```
대사:
  "근데 블로거 퀄리티가 낮지 않아요.
   슈퍼멤버스는 등급제라서
   자연스럽게 상위 블로거만 오거든요."

텍스트:
  "비싸다고 퀄리티 높은 게 아님"
  "구조가 퀄리티를 만든다"
```

**[27~30초] CTA**
```
대사: "지금 무료로 먼저 해보세요."
텍스트: "👇 무료 체험 → 카드 없음"
```

## 기획안 4 — "이번 달 폐업 신고서" (감정 서사)

**가설**: H4  
**예상 CTR**: 11~15%, 예상 CPL: 15,000~20,000원  
**길이**: 35초

### 스크립트

**[0~2초] 훅**
```
화면: 텅 빈 저녁 매장. 사장님 혼자 테이블.
텍스트: "이번 달 폐업 신고서 뽑아봤습니다"
사운드: 빗소리 → 뚝 끊김
```

**[2~15초] 위기**
```
대사:
  "임대료 350.
   직원 월급 280.
   이번 달 매출... 480.
   계산해보세요."

텍스트 순차 등장:
  "지출: 630만원"
  "매출: 480만원"
  "💸 -150만원"
```

**[15~28초] 반전**
```
화면: 폰 알림 쏟아짐
대사:
  "슈퍼멤버스 쓰고 한 달 뒤에
   블로거 리뷰가 쌓이면서
   네이버에서 우리 가게가 보이기 시작했어요.
   예약이 차기 시작했고요."

텍스트:
  "리뷰 +31개 (30일)"
  "네이버 순위 ↑"
  "이번 주 예약 🔴 마감"
```

**[28~35초] CTA**
```
대사:
  "폐업 생각하기 전에 이거 먼저 해보세요.
   월 15만원이에요."
텍스트: "👇 무료로 시작 (카드 없음)"
```

## 기획안 5 — "사장님 잘못 아닙니다" (자책 → 희망)

**예상 CTR**: 9~12%  
**길이**: 40초

### 스크립트

**[0~3초] 훅**
```
화면: 사장님, 진지하게 카메라 보며
대사: "손님이 없는 거, 사장님 잘못 아닙니다."
텍스트: "문제는 맛이 아니었어요"
```

**[3~20초] 진단**
```
대사:
  "더 맛있게 만들려고 레시피 바꿨고,
   인테리어도 손봤고, 가격도 낮췄는데.
   
   그래도 손님이 없었어요.
   
   알고 보니까 네이버에서 우리 가게가 안 보였던 거예요.
   네이버 플레이스 순위가 낮으면
   아무도 찾질 못해요."

텍스트:
  "레시피 개선 → ❌"
  "인테리어 → ❌"
  "가격 인하 → ❌"
  "진짜 문제: 네이버에서 안 보임"
```

**[20~33초] 해결**
```
화면: 흑백 → 컬러 전환
대사:
  "슈퍼멤버스로 진짜 손님 리뷰 쌓았더니
   한 달 만에 네이버 검색에서 보이기 시작했어요.
   
   음식은 처음부터 좋았어요.
   그냥 아무도 못 찾은 거예요."

텍스트:
  "한 달 후: 네이버 상위 노출"
  "음식은 처음부터 좋았다"
```

**[33~40초] CTA**
```
대사: "이제 알릴 수 있어요. 무료로 시작해보세요."
텍스트: "👇 무료 시작"
```
