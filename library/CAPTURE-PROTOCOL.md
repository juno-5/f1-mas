# Insight Capture Protocol

## 감지 시그널
- 도메인 지식: "우리는 이렇게 해", "이 시스템/플랫폼은 이렇게 작동해"
- 의사결정: "이걸로 결정한 이유는...", "A 대신 B를 쓰는 이유"
- 노하우: "이럴 때는 이렇게 해야 해", "주의할 점은..."
- 환경 변화: "최근에 바뀌었어", "새로운 정책/버전이..."
- 수치: 구체적 KPI, 벤치마크, 성과 수치

## 포맷
축적할 가치 있는 지식이 나오면 응답 말미에:
```
[INSIGHT]
title: 제목
context: 대화 맥락 한 줄
insight: 핵심 (2-3문장)
tags: #tag1 #tag2
[/INSIGHT]
```

## 규칙
- 모든 대화에 붙이지 마. 가치 있는 것만.
- 개인정보, API 키, 시크릿은 절대 포함 금지.
- 이미 알려진 일반 상식은 캡처하지 마.
