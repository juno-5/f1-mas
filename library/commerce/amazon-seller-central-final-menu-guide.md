# I. 메인 메뉴 구조 (5가지)

Amazon Seller Central의 메인 메뉴는 **Home 페이지의 링크**로 구성:

## 📊 1. 판매 대시보드 (Sales Dashboard)
**URL**: `https://sellercentral-japan.amazon.com/business-reports/#/dashboard`

### 페이지 구성

```
┌─────────────────────────────────────────┐
│ HEEDA | 일본 | 신규 셀러 센트럴          │
├─────────────────────────────────────────┤
│ 보고서 메뉴 닫기                          │
│                                         │
│ ✓ 대시보드 (현재)                        │
│ ✓ 판매 대시보드                          │
│                                         │
│ ━━ 비즈니스 보고서                       │
│   ├─ 날짜별                             │
│   │  ├─ 판매 및 트래픽                  │
│   │  ├─ 판매 및 트래픽 상세 페이지      │
│   │  └─ 셀러 퍼포먼스                    │
│   │                                    │
│   └─ ASIN별                            │
│      ├─ 판매 및 트래픽 상세 페이지      │
│      ├─ 상위 아이템별 판매 및 트래픽     │
│      └─ 하위 아이템별 판매 및 트래픽     │
│                                         │
└─────────────────────────────────────────┘
```

### 주요 섹션

#### A. 판매 대시보드 (Sales Dashboard)
**시간별 판매 트렌드**:
- 00:00 ~ 23:00까지 24시간 시간대별 차트
- "주문 수량" 그래프
- "주문 상품 판매량" 그래프

**판매 요약**:
```
생성: 2026.2.24. 오후 1시 37분 47초 GMT+9

총 주문 아이템:    0개
주문 수량:         0개  
주문 상품 판매량:  ¥0.00
평균 수량/주문:    0
평균 판매량/주문:  ¥0.00
```

**비교 섹션** (당일 vs 어제 vs 지난주 같은요일 vs 지난해 같은날):
```
당일 합계(현재까지): 0 상품, ¥0.00
어제(당일저녁):     5 상품, ¥11,800.00
지난주 같은요일:    3 상품, ¥7,100.00
지난해 같은날:      0 상품, ¥0.00
```

#### B. ASIN 퍼포먼스 심층 분석 (Deep Dive)
```
기간: 2026.2.22.에 끝나는 월-일 주 vs 이전 월-일 주

상품명: HEEDA 炭オールインワン クレンジングバーム
ASIN: B0D7C63549
OPS 증가: ¥1,878
```

│ ├─ FBA 주문 보기                        │
│ └─ 빠른필터:                           │
│    ├─ 당일배송                         │
│    ├─ 프리미엄 미배송                   │
│    ├─ 비즈니스 고객 미배송              │
│    ├─ 배송 지연 직전                    │
│    └─ 취소 직전                        │
│                                        │
│ 세부 분류:                              │
│ ├─ 주의 요망                           │
│ ├─ 취소 직전                           │

│ 다양한 필터 옵션:                       │
│ ├─ 발송 기한 (모든 날짜, 당일, 익일)   │
│ ├─ 판매 채널 (Amazon.co.jp)            │
│ ├─ 배송 서비스 (프리미엄)               │

│                                        │

│ ├─ 주문 날짜 (오름/내림)                │

│ ├─ 배송 서비스 (오름/내림)              │
│ └─ 상태 (오름/내림)                    │
│                                        │
│ 페이지당 결과: 15 / 25 / 50 / 100     │
│                                        │

# II. 메뉴 간 네비게이션

## Home 페이지에서 이동 가능한 모든 링크

```
[판매 대시보드로 이동]
  ↓
  Business Reports Dashboard
  ├─ 시간별 판매 트렌드
  ├─ ASIN 퍼포먼스 분석
  └─ 서브메뉴: 비즈니스 보고서
     ├─ 날짜별 (판매, 트래픽, 성과)
     └─ ASIN별 (세부 분석)

[미결 주문으로 이동]
  ↓
  Orders Management
  ├─ 주문 검색 & 필터
  ├─ 상태별 분류
  └─ 배송 설정 관리

[피드백 관리자로 이동]
  ↓
  Feedback Manager
  ├─ 별점 분석
  ├─ 서비스 인사이트
  └─ 최근 피드백 보기

[모든 재고 관리로 이동]
  ↓
  Inventory Management
  ├─ 상품 목록 (9개)
  ├─ 상태별 필터
  │  ├─ 초안 (1)
  │  ├─ 활성화 (1)
  │  ├─ 최적화 필요 (4)
  │  └─ 검토 중
  └─ 상품 상세 정보
     ├─ 가격
     ├─ 재고
     └─ 판매 순위

[리스팅 광고] (상품별)
  ↓
  Advertising Console
  ├─ SP 광고 설정
  └─ 키워드 & 입찰가 관리

[퍼포먼스 분석] (상품별)
  ↓
  Business Reports
  └─ 해당 상품의 상세 데이터

[해외 리스팅 관리]
  ↓
  Global Selling
  └─ 다른 마켓 리스팅 관리
```

# III. 각 메뉴의 액션 가능한 작업

| 메뉴 | 주요 기능 | 액션 |
|------|---------|------|
| **Sales Dashboard** | 시간별 판매, ASIN 성과 | 데이터 분석만 (읽기) |
| **Orders** | 주문 검색, 필터, 추적 | 배송 처리, 취소, 반품 관리 |
| **Feedback Manager** | 별점, 서비스 지표, 최근 피드백 | 피드백 응답, 거래 상세 조회 |
| **Inventory** | 상품 목록, 상태 분류 | 가격 변경, 재고 수정, 리스팅 편집 |
| **Advertising** | 캠페인, 성과, 광고 도구 | 캠페인 생성/편집, 입찰가 관리, 보고서 |

# IV. 메뉴 사용 타이밍 (Best Practice)

## 📋 일일 (Daily)
```
09:00 - Home (5분)
        └─ Global Snapshot 확인 (판매, 주문, 피드백)

17:00 - Orders (10분)
        └─ 새 주문 확인, 배송 처리

19:00 - Feedback Manager (5분)
        └─ 새 피드백 확인
```

## 📊 주간 (Weekly)
```
월요일 - Sales Dashboard (20분)
         └─ 지난주 vs 이번주 비교, 트렌드 분석

수요일 - Advertising (15분)
         └─ 캠페인 ACOS 검토, 최적화

금요일 - Inventory + Feedback (20분)
         ├─ 재고 부족 상품 확인
         └─ 주간 피드백 패턴 분석
```
